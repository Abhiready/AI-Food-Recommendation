# AI Food Recommender

An intelligent food ordering and recommendation system built with Python, Flask, and Scikit-learn. This application provides personalized restaurant suggestions based on user preferences and implements a complete user authentication system with a MySQL database.

# Features

-   Content-Based Recommendations: Suggests restaurants based on cuisine, type, and collections.
-   *Dual Recommendation Modes:*
    1.  Find restaurants similar to a user's favorite choice.
    2.  Recommend based on a free-text query (e.g., "cheesy pizza" or "healthy salad").
-   *Secure User Authentication:* Full registration and login system with password hashing.
-   *Protected Routes:* The main recommendation engine is only accessible to logged-in users.
-   *Dynamic Frontend:* A clean, responsive, and animated user interface built with Bootstrap 5.

# Tech Stack

-   *Backend:* Python, Flask, Flask-SQLAlchemy, Flask-Login
-   *Database:* MySQL
-   *Machine Learning:* Scikit-learn, Pandas, NumPy
-   *Frontend:* HTML, CSS, Bootstrap 5, Jinja2
